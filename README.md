# Jogo-Golden-Godot
Jogo plataforma 2D criado no Godot para um projeto do segundo semestre da faculdade de ciência da computação

*Funcionalidades:
implementação lógica de jogo e interações básicas
Tipo de jogo plataforma
Implementação de movimentos básicos
Sistema de colisão 
Mecânicas de ataque variado
Sistema de barra de vida
